## Usage Example

After adding these resources to both English and Arabic files:

```razor
<!-- Instead of: -->
<h5>Import Products from Excel</h5>

<!-- Use: -->
<h5>@ECommerceResources.ImportProducts</h5>

<!-- Instead of: -->
<button>Download Template (With Reference Data)</button>

<!-- Use: -->
<button>@ECommerceResources.DownloadTemplateWithData</button>
```

## Build Steps

1. Add all resources to both `.resx` and `.ar.resx` files
2. Rebuild solution
3. Test import page with different language settings
4. Verify dropdown lists work correctly
5. Test Excel template generation and parsing

## Notes

- **Dropdowns** now show actual names (Arabic/English based on current language)
- **User-friendly**: No need to remember GUIDs
- **Auto-fit columns**: Excel columns adjust to content width
- **Validation**: Accepts both names and IDs for backward compatibility
