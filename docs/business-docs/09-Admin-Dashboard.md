# لوحة تحكم الإدارة (Admin Dashboard)

## الوظائف الأساسية

### إدارة البائعين:
- الموافقة على البائعين الجدد
- إيقاف/تفعيل الحسابات
- مراجعة الأداء
- التحقق من الوثائق
- معالجة طلبات البائعين
- مراقبة النشاط المشبوه

### إدارة الكتالوج:
- مراجعة المنتجات الجديدة
- تعديل المنتجات الرئيسية
- إدارة الفئات والتصنيفات
- إزالة المخالفات
- إدارة العلامات التجارية
- مراجعة المحتوى

### إدارة الطلبات:
- مراقبة جميع الطلبات
- حل النزاعات
- إدارة المرتجعات
- استرداد الأموال
- تتبع الشحنات
- معالجة الشكاوى

### إدارة المحتوى:
- الصفحة الرئيسية
- البانرات والإعلانات
- الصفحات الثابتة
- المدونة والمقالات
- الحملات الترويجية
- إدارة الـ Merchandising

### التقارير والإحصائيات:
- GMV (Gross Merchandise Value)
- عدد الطلبات
- أداء البائعين
- معدلات التحويل
- معدلات الإرجاع
- تحليلات العملاء

## مؤشرات الأداء الرئيسية (Admin KPIs)

| المؤشر | الوصف | الهدف |
|--------|-------|--------|
| GMV | إجمالي قيمة البضائع المباعة | زيادة شهرية 15% |
| Active Sellers | عدد البائعين النشطين | +50 شهرياً |
| Active Customers | عدد العملاء النشطين | +500 شهرياً |
| Order Fulfillment Rate | نسبة الطلبات المكتملة | >95% |
| Dispute Rate | نسبة النزاعات | <2% |
| Average Order Value | متوسط قيمة الطلب | زيادة 10% شهرياً |
| Customer Satisfaction | رضا العملاء | >4.5/5 |
