🎛️ 14. لوحة تحكم الإدارة

14.1 🔧 الوظائف الأساسية

إدارة البائعين:

  • الموافقة على البائعين الجدد
  • إيقاف/تفعيل الحسابات
  • مراجعة الأداء
  • التحقق من الوثائق

إدارة الكتالوج:

  • مراجعة المنتجات الجديدة
  • تعديل المنتجات الرئيسية
  • إدارة الفئات
  • إزالة المخالفات

إدارة الطلبات:

  • مراقبة جميع الطلبات
  • حل النزاعات
  • إدارة المرتجعات
  • استرداد الأموال

إدارة المحتوى:

  • الصفحة الرئيسية
  • البانرات
  • الصفحات الثابتة
  • المدونة

التقارير والإحصائيات:

  • GMV (Gross Merchandise Value)
  • عدد الطلبات
  • أداء البائعين
  • معدلات التحويل
  • معدلات الإرجاع

14.2 📊 مؤشرات الأداء الرئيسية (Admin KPIs)

المؤشر الوصف الهدف
GMV إجمالي قيمة البضائع المباعة زيادة شهرية 15%
Active Sellers عدد البائعين النشطين +50 شهرياً
Active Customers عدد العملاء النشطين +500 شهرياً
Order Fulfillment Rate نسبة الطلبات المكتملة >95%
Dispute Rate نسبة النزاعات <2%

14.3 🔐 مستويات الصلاحيات

الدور الصلاحيات
Super Admin كل الصلاحيات + إدارة المستخدمين
Content Manager إدارة المحتوى والكتالوج
Seller Manager إدارة البائعين والموافقات
Customer Support إدارة التذاكر والنزاعات
Finance Manager التقارير المالية والمدفوعات
