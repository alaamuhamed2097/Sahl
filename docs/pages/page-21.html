🎫 أنواع أكواد الخصم (Promo Codes)

🟩 1) Promo Code Based on Price

مثال: "خصم 10% على أي منتج سعره فوق 1000"

من يعمل البرومو؟

  • ✔ المنصة (Platform Promo)

من يتحمل التكلفة؟

  • ✔ المنصة 100%

🟦 2) Promo Code Based on Category

مثال: "خصم 15% على قسم الإلكترونيات فقط"

من يعمل البرومو؟

  • ✔ المنصة

من يتحمله؟

  • ✔ المنصة بالكامل

🟧 3) Promo Code Based on Brand

مثال: "خصم 10% على جميع منتجات Samsung"

من يعمل البرومو؟

  • ✔ المنصة
  • ✔ أو وكيل العلامة

من يتحمل التكلفة؟

  • ✔ المنصة
  • ✔ أو الوكيل / المورد
  • ✔ أو Co-Funded بين الطرفين

🟨 4) Promo Code Based on Vendor

مثال: "خصم 10% على منتجات متجر ABC"

من يعمل البرومو؟

  • ✔ البائع أو المنصة

من يتحمل التكلفة؟

  • لو البائع أنشأ الكود → البائع يتحمل الخصم
  • لو المنصة أنشأته → المنصة تتحمل الخصم

🟫 5) Promo Code على مجموعة منتجات

مثال: "10% على منتجات Back To School"

من يعمل البرومو؟

  • ✔ المنصة

من يتحمل الخصم؟

  • ✔ المنصّة 100%
  • أو ✔ Co-Funded من الباعة (نادر)

🟥 6) Promo Code للعميل الجديد Only

مثل Noon: NOONNEW, FIRST10, NEW50

من يعمل البرومو؟

  • ✔ المنصة فقط

من يتحمل التكلفة؟

  • ✔ المنصة بالكامل

لأن هدفه زيادة Acquisition وليس دعم البائعين.

🧩 الخلاصة

نوع البرومو من ينشئه؟ من يتحمل الخصم؟
السعر أكبر من 1000 المنصة المنصة
Category المنصة المنصة
Brand المنصة/الوكيل المنصة أو الوكيل أو الاثنين
Vendor Promo البائع البائع
Platform Promo المنصة المنصة
Co-Funded Promo المنصة المنصة + البائع
New User Promo المنصة المنصة

🎉 الخلاصة النهائية

قاعدة عامة:

أي Promo Code مبني على شروط عامة (السعر – الفئة – العلامة – الحد الأدنى للطلب) → المنصة هي المسؤولة عنه وتموله

لأن البائع لا يقدر يتحمل خصم "تلقائي" على آلاف المنتجات اللي مش بتاعته.
