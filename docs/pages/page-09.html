🟨 4. دورة حياة تذكرة الدعم (Ticket Life Cycle)

الإنشاء

Created

المستخدم يفتح تذكرة

الاستلام

Acknowledged

المنصة تستقبل الطلب وتحدّد الأولوية

التوجيه

Assigned

إلى فريق العملاء أو البائعين أو الفريق الفني

الرد الأولي

First Response

يتم خلال SLA

المعالجة

In Progress

إما حل مباشر أو طلب مستندات أو تواصل مع البائع/العميل

التصعيد

Escalated

لو المشكلة معقدة → تذهب لـ Level 2 أو 3

الحلّ

Resolved

إرسال الحل للمستخدم

الإغلاق

Closed

بعد موافقة الطرف أو بعد فترة زمنية
