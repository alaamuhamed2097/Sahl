🔷 4. عرض البائع للعميل (Customer Facing)

4.1 العرض الفائز (Buy Box Offer)

هو العرض الذي يظهر للعميل كخيار رئيسي في صفحة المنتج.

يتم اختياره بناءً على:

  • ✅ السعر الأفضل
  • ⚡ وقت الشحن الأسرع
  • ⭐ تقييم البائع
  • 📦 توفر المخزون
  • 📊 سجل الإلغاء والتأخير
  • 🎯 اشتراك العرض في الحملات

4.2 باقي العروض (Other Sellers Section)

يتم عرض جميع العروض الأخرى في قسم منفصل يحتوي على:

البيان الوصف
اسم البائع Seller Name + Rating
السعر Price + Discount
تكلفة الشحن Shipping Cost
وقت التجهيز Handling Time
حالة المنتج New / Used / Refurbished
مدة الضمان Warranty Period
📌 هذا مشابه تمامًا لأقسام:
  • Other Sellers on Amazon
  • More Offers from Noon Sellers

🔷 5. واجهة إدارة البائع (Seller Dashboard)

البائع من خلال لوحة التحكم يستطيع:

📝 إدارة العرض

  • إضافة عرض جديد لنفس المنتج
  • تعديل السعر
  • تعديل المخزون
  • تغيير وقت التجهيز
  • ضبط تكلفة الشحن
  • تغيير حالة المنتج
  • تحديث الضمان
  • تفعيل/إيقاف العرض

📊 متابعة الأداء

  • متابعة احتمالية الفوز بالBuy Box
  • مقارنة سعره بأسعار المنافسين
  • تحليل المبيعات
  • مراقبة التقييمات

🔷 6. قواعد العمل (Business Rules)

Rule 1

لا يمكن للبائع تعديل معلومات المنتج الأساسية (الوصف – العلامة – المواصفات).

يستطيع تعديل بيانات عرضه فقط.

Rule 2

العروض التي مخزونها = 0 تتحول تلقائيًا إلى Out of Stock.

Rule 3

العروض ذات الحالة "Open Box / Used / Refurbished" لا تنافس عروض "New" في الـ Buy Box إلا إذا لم يوجد عروض جديدة.

Rule 4

العرض ذو تقييم بائع منخفض لا يمكن أن يفوز بالBuy Box مهما كان السعر.

Rule 5

العرض يتم تعطيله تلقائيًا عند:

  • مخالفات
  • تأخيرات شحن مرتفعة
  • شكاوى كثيرة
  • بيانات غير كاملة

Rule 6

عروض البائع تظهر فقط لو تم توثيق البائع (KYC Verified).

Rule 7

لكل عرض سجل تاريخي (Offer History) للنقاط:

  • تغييرات الأسعار
  • تغييرات المخزون
  • تغييرات الشحن
  • تغييرات الحالة
